# User Documentation

## Log In

Click on Log-In to bebredirected to your SSO provider login page.

## Dashboard page and components

This dashboard displays the summaries of the expenses, like the metadata (total, daily expenses) and charts for expenses, by user and by country.

The provided form allows filtering out exceptional expenses and define a start and end date to the displayed expenses.

This dashboard is the same in the countries pages. The only difference is that it's firstly filtered for the selected country.

## Management pages

Management pages allow you to add the different items you need to manage your expenses.

### User

Adds a simple username to register the expenses under their name.

### Currency

The currencies you'll use for your expenses and to associate with countries.

Add the label, the fullname, the locale code for the format ([list is found here](https://www.w3schools.com/tags/ref_language_codes.asp)) and the conversion ration to your main currency (1.00 for your main currency).
